# Namoshi Documentation

This repository contains the official documentation for the Namoshi Protocol, the decentralized identity service for the Bitcoin ecosystem on Citrea.

## Structure

- `protocol/`: Core concepts and architecture of the naming service.
- `resolution/`: Technical guides for resolving names and avatars.
- `guides/`: UI/UX integration guides for wallets and leaderboards.

## Development

To preview the documentation locally, install the Mintlify CLI:

```bash
npm i -g mintlify
```

Then run the development server:

```bash
mintlify dev
```
